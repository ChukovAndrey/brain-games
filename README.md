# Brain-games (console games written in javascript)
## Installation
```
git clone https://github.com/ChukovAndrey/frontend-project-lvl1.git
```
```
cd frontend-project-lvl1
```
```
make install
```
```
sudo npm link
```
To start a game use commands listed below:
```
brain-calc
```
```
brain-even
```
```
brain-gcd
```
```
brain-prime
```
```
brain-progression
```
